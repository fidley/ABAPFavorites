ABAP Favorites

The plugin contains two separate views Favorites and Favorties Dev Objects. The difference between them, is that in Favorites view you can create two kind of folders and in Favorites Dev Objects view you can only create folders for Development Objects (programs, classes, interfaces, etc.). This distinguish is done because some of persons does want to store this favorites separately. If you don’t want that then it’s enough to use Favorites view.
