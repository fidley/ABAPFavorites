ABAP Favorites - Favorites View

Favorites folders

Name, Description, Long Description

Name and description will be visible on favorites list. It will be also used for filtering. Long description will be only visible at edit.

Project independent?

When selected, it will make that doesn’t matter on which SAP system you have currently active editor/GUI, the folder will be visible. If you’ll make your folder project dependent (unselected) then it will appear on the list only if your active editor/GUI is linked to the same SAP system.
Additionally, when the folder is project dependent, the favorite object will be always run/open in that project (SAP system)

Development Objects Folder?

There are two type of folders available:

  • Standard one
  • For Development Objects (checkbox selected)

Standard folders can only be created in Favorites view, and they can contain only following objects:

  • Transactions
  • Programs
  • URLs
When you double-click on one of this objects then they will be run in new tab of SAP GUI.

Development Objects folder can be created in both Favorites and Favorites Dev Objects views. It can contain following development objects:

  • Programs
  • Classes
  • Interfaces
  • Function Groups
  • Function Modules
  • Tables
  • Views
  • Message Classes
  • Search Helps
  • ADT Links
  • CDS Views
  • Packages
When you double-click on one of this objects then they will be opened in new Editor tab.

New Favorite entry

When you're adding new objects, you can see two kinds of dialogs.

New entry for ABAP objects

For most of the objects you'll be able to choose Name (object name in SAP), description, long description and shortcut command (one of 10)

New entry for URL and ADT link objects

For URLs and ADT Links, name, description and long description can be fill freely, important is to fill correctly the URL of the entry.

Commands

It is possible to bind the favorite to a key combination, which can be used to run or open favorite object.
When the folder containing object is linked to a certain project, it will be used as project in which the favorite will be run/open. In other case, current active ADT project will be used.
There are 10 key bindings foreseen.

Additional view menu functions

Export/Import

Can be used to export favorites to XML file, or to import favorites from XML file.
