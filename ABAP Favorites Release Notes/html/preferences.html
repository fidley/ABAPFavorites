ABAP Favorites - Preferences

Plugin preferences

Hide project dependent folders

When selected, project dependent folders will be only shown, when the project for active ABAP editor/SAP GUI will be matching the project linked to folder.

Save state of folder expansion

When selected will save the expansion state of the folders in the view.

Navigate to eclipse for supported development objects

Project to be used during double click

Determines behaviour of double click. You can decide if on double click object will be open/run in active project (or linked in case of project dependent ones) or if the popup window with the project list will be shown.

Project to be used during double click when pressed Ctrl Key

Determines behaviour of double click when pressing Ctrl key. You can decide if on double click object will be open/run in active project (or linked in case of project dependent ones) or if the popup window with the project list will be shown.

Key bindings for favorites commands

Key combinations are predefined, but they can be adapted in the Preferences (General->Keys)
